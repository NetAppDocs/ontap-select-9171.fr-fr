Il existe plusieurs tâches connexes que vous pouvez effectuer dans le cadre de la sécurisation d'un déploiement ONTAP Select .



= Modifier le mot de passe de l'administrateur de déploiement

Vous pouvez modifier le mot de passe du compte administrateur de la machine virtuelle Deploy selon vos besoins à l’aide de l’interface de ligne de commande.

.Étapes
. Sign in à l’interface de ligne de commande de l’utilitaire de déploiement à l’aide du compte administrateur.
. Changer le mot de passe :
`password modify`
. Répondez à toutes les invites en fonction de votre environnement.
